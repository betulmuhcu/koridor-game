# koridor-game
HTML5 ve JavaScript ile geliştirilmiş minimalist bir koşu oyunudur


**Koridor**, reflekslerinizi test eden, engellerden kaçıp portallardan geçerek en yüksek skoru hedeflediğiniz web tabanlı, 2D bir "hyper-casual" kaçış oyunudur. 

## 🎮 Özellikler
* **Akıcı Oynanış:** Tek dokunuş/tuş ile çalışan şerit değiştirme mekaniği.
* **Dinamik Zorluk:** Oyunda ilerledikçe artan hız ve sıklaşan engeller.
* **Ses ve Görsel Efektler:** AudioContext API ile anlık üretilen 8-bit tarzı sesler ve HTML5 Canvas ile parçacık (particle) patlama efektleri.

## 🛠️ Kullanılan Teknolojiler
* **HTML5 Canvas:** Oyun içi grafikler ve render işlemleri.
* **CSS3:** Modern neon arayüz tasarımı, menü animasyonları ve responsive yapı.
* **Vanilla JavaScript:** Oyun motoru, çarpışma algılama (collision detection), fizik kuralları ve durum yönetimi.

## 🕹️ Nasıl Oynanır?
* **Masaüstü:** Şerit değiştirmek için Space tuşunu kullanın.
* **Mobil:** Şerit değiştirmek için ekrana dokunun.
* **Amaç:** Engellere (çukurlar ve dikenler) çarpmadan en uzun mesafeye ulaşın.
